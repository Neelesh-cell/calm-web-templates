Focus Management

For users with ADHD or cognitive load sensitivities, losing track of where you are on a page can be frustrating. Navigate this page using the Tab key. Notice the prominent, high-contrast focus outlines ensuring you always know where you are.

How it works: We use :focus-visible with custom outlines that exceed default browser styling.

Animation Control

For users with vestibular disorders or sensory sensitivities, unexpected motion can be genuinely distressing — not just annoying. Toggle "Calm Mode" to pause these elements immediately.

How it works: Respects prefers-reduced-motion automatically before you even toggle.
