# calm-web-templates
UI templates focusing on low-stimulation environments, readable typography, and strict semantic HTML.
This repository serves as a functional demonstration of accessibility-first development for neurodivergent users. Built entirely with semantic HTML, CSS custom properties, and Vanilla JavaScript, it prioritizes cognitive load reduction and user control without the overhead of heavy frameworks. Core implementations include dynamic root-level text scaling, instant reduction of CSS animations, and a persistent dark/light theme toggle that prevents visual flash on render.
